PE-Agent: Multimodal Pulmonary Embolism Intelligent Diagnosis System
Project Overview

PE-Agent is a multimodal AI-powered assistant system for Pulmonary Embolism (PE) diagnosis support, integrating:

CT imaging analysis
Structured clinical data
Chinese clinical text (NLP)
Explainable AI (Grad-CAM)
Automated diagnostic report generation
Key Highlights
Multimodal fusion (Imaging + Tabular + Clinical Text)
Chinese medical NLP (BERT-based models)
Explainable AI (Grad-CAM visualization)
AI-driven diagnostic report generation (Agent-based workflow)
Supports private medical data (privacy-preserving design)
Data Description

Due to privacy constraints, real clinical data is not publicly released.

Uses private Chinese clinical datasets
Sample data is available in data/sample/
Alternative public datasets can include:
RSNA Pulmonary Embolism Dataset
MIMIC-IV Clinical Database
